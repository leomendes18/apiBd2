API para registrar e ver as localizações de hemocentros

    Para adicionar um hemonucleo no mapa vá para a seção "Salvar Hemonucleos", clique no lugar que quer adicionar no mapa, adicione um nome e clique salvar.

    Para ver os hemonucleos vá para a seção "Encontrar Hemonucleos" e clique no botão Listar, informar um nome para a pesquisa é opcional.
